font-size : 글꼴의 크기를 변경할 때 사용

        선택자{font-size : 크기(px,em(배수)%,)}
  • 고정크기 px텍스트
  • 가변크기 em텍스트
  • 가변크기 %텍스트

폰트의 두께, font-weight : 글꼴의 굵기를 표현

        선택자{
            font-weight : lighter|normal|bold|bolder|100|400|900
        }
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant : 영문 텍스트문구를 작은 대문자로 변경할 때 사용

        선택자{font-variant:small-caps;}

font-style : 텍스트 문구를 기울이고자 할 때

        선택자{font-style:normal|italic|oblique}

italic 기울임 글꼴

oblique 원래 글자를 기울인셈
